# CALCULATOR APP USING NODEJS (beginner)

## Details

This is a simple calculator app project Node.js app.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Square
- Square Root
- Cube

## Tech Stack

- Language - Node.js/Express(ESM)

## Dependences

colors - For aesthetics
nodemon - To ensure that the server continues to run even if edits are being made

## Git Repository

https://github.com/MyITjournal/Calculator.git

## Installation Steps

1. Create a folder where you want the project to be installed.

2. Navigate to that folder, for example:

`cd Calculator`

3. Clone the repository.

git clone https://github.com/MyITjournal/SampleProject1.git

4. Install the dependences.

`npm install`

5. Start the server

`npm start`

Production URL: http://localhost:3000/
